# STYLZ Career AI – Resume Optimizer

## Overview
AI-powered resume optimization tool that analyzes resumes against job descriptions and generates improved, tailored versions.

## Key Features
- Resume and job description input
- AI-powered resume enhancement using OpenAI API
- Match scoring and optimization suggestions
- Copy/download optimized resume output

## Tech Stack
React | Node.js | Express | OpenAI API | JavaScript

## SDLC Approach

**Planning**
- Identified problem of resume-job mismatch
- Defined core features and user flow

**Design**
- Designed frontend UI and backend API structure
- Planned OpenAI integration

**Development**
- Built React frontend for user interaction
- Developed backend API for resume processing
- Integrated OpenAI API

**Testing**
- Tested API responses and UI functionality
- Handled edge cases and errors

**Deployment**
- Deployed frontend (Vercel) and backend (Render)
